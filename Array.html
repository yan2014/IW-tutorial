Definition

Series are used to store multiple values in a single variable, and avoid the effort of defining multiple variables. It is comparable to arrays in JavaScript and lists in Python.

Example

In this example, x, y, and z, are variables:

var x = 5;

var y = 6;

var z = x + y;
